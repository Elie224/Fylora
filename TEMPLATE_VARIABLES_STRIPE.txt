=== NOTE IMPORTANTE ===
⚠️  Vous avez un Product ID (prod_...), mais nous avons besoin du Price ID (price_...)

Pour trouver le Price ID:
1. Allez sur la page du produit dans Stripe Dashboard
2. Cherchez la section "Tarifs" ou "Pricing"
3. Le Price ID est affiché là (commence par price_...)

- Cliquez sur "Reveal test key"

- Stripe Dashboard > Developers > Webhooks
- Cliquez sur votre endpoint
- Cliquez sur "Reveal" pour le Signing secret

- Cliquez sur chaque produit
- Copiez le Price ID (commence par price_...)
